# Caso de Uso: Atualizar Dados do Administrador

## **Atores Envolvidos**
- **Administrador**

---

## **Pré-condições**
- O administrador deve estar conectado à internet.
- O administrador deve estar cadastrado no sistema.

---

## **Pós-condições**
- Os dados do administrador devem estar atualizados no sistema.

---

## **Fluxo Principal - Atualizar Dados**
1. O administrador acessa a tela de atualização de dados.  
2. O administrador informa os dados que deseja atualizar.  
3. O sistema busca o administrador pelo **ID** informado.  
4. O sistema atualiza os dados do administrador no banco de dados.  
5. O sistema redireciona o administrador para a área principal do sistema.
